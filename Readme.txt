■実装されているアルゴリズム
①盤面と同じ数の評価値表を作成
②左上から順にマスを洗っていく、まず打てるマスか判定(打てないマスの場合評価値に-1024を入れて次へ)
③打てるマスの場合、「仮にそこに打ったら何枚ひっくり返せるか」を一旦そのマスの評価値とする(なお、四隅のマスだった場合評価値を5足しておく)
④盤面のコピーを作成
⑤打ったと仮定した盤面でコピーを更新、手番を相手にし、相手が一番ひっくり返せるマスに打ったとして、その枚数を評価値から引く
⑥またコピーを更新、手番を自分にし、一番ひっくり返せるマスに打ったとして枚数を評価値に足し、盤面を更新する
⑦⑤～⑥を決められた回数繰り返す(パスの場合は何もしないで相手の手番にする)
⑧そうしてできた評価値表を左上から洗っていき、最高の評価値を探索する
⑨そのマスに打つ
